# Security Policy

## Reporting A Vulnerability

Please do not open a public issue for a suspected vulnerability.

Email security reports to the FerricStore maintainers. Include:

- affected package and version;
- reproduction steps;
- impact and expected behavior;
- any suggested fix or mitigation.

We will acknowledge valid reports and coordinate disclosure timing before publishing details.

## Supported Versions

The SDK is currently pre-1.0. Security fixes target the latest released minor version unless a maintainer states otherwise.
